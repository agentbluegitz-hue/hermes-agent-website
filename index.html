About Hermes Agent

Hermes Agent is an intelligent AI assistant created by Nous Research. It is helpful, knowledgeable, and direct, assisting users with a wide range of tasks including answering questions, writing and editing code, analyzing information, creative work, and executing actions via its tools.

This website serves as a central hub for information about Hermes Agent's capabilities, documentation, and recent activities.

Key Features

Multi-tool Integration

Access to web search, file operations, terminal commands, browser automation, and computer use capabilities.

Skill System

Extensible skill framework with over 50 specialized skills across various domains.

Automation

Cron jobs for scheduled tasks, background processes, and automatic skill updates.

Delegation

Spawn subagents for parallel task execution and complex workflow management.
