# Hermes Agent Website

This website showcases the Hermes Agent AI assistant - its capabilities, skills, activities, and documentation.

## About Hermes Agent

Hermes Agent is an intelligent AI assistant created by Nous Research. It is helpful, knowledgeable, and direct, assisting users with a wide range of tasks including answering questions, writing and editing code, analyzing information, creative work, and executing actions via its tools.

## Features

- **Multi-tool Integration**: Access to web search, file operations, terminal commands, browser automation, and more
- **Skill System**: Extensible skill framework for adding new capabilities
- **Automation**: Cron jobs for scheduled tasks and background processes
- **Version Control**: Git integration for skills and configurations
- **Computer Use**: Background desktop automation capabilities
- **Delegation**: Spawn subagents for parallel task execution

## Skills Categories

Hermes Agent skills are organized into categories:

- **autonomous-ai-agents**: Spawning and orchestrating AI coding agents
- **computer-use**: Background desktop automation
- **creative**: ASCII art, diagrams, design tools
- **data-science**: Jupyter notebooks and data analysis
- **email**: Email management via terminal
- **github**: GitHub workflow management
- **hermes-file-operations**: Safe file operations
- **media**: GIF search, YouTube transcripts, music generation
- **mlops**: Machine learning operations tools
- **note-taking**: Obsidian integration
- **productivity**: Document creation, presentations, spreadsheets
- **research**: Academic research, paper discovery
- **smart-home**: Philips Hue light control
- **social-media**: X/Twitter integration
- **software-development**: Debugging, testing, code review
- **yuanbao**: User grouping and mentions
- **zotero-library-organizer**: Zotero reference management

## Recent Activities

The Hermes Agent continuously learns and improves through:
- Daily AI briefing system with Arxiv paper detection
- Automated skill version control with hourly commits
- Zotero library organization with automated Arxiv paper filing
- Continuous integration and deployment workflows

## Getting Started

To use Hermes Agent:
1. Install the Hermes Agent software
2. Configure your API keys and preferences
3. Load skills as needed for your tasks
4. Start chatting and delegating tasks!
